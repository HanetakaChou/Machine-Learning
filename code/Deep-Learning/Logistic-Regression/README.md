### Activation Function  

Why non-linear activation is required?  

Universal Approximation Theorem  

A neural network with at least one hidden layer and non-linear activation functions can approximate any continuous function to a desired degree of accuracy, given sufficient neurons.  

In contrast, a network with only linear activation functions lacks this expressive power and can only represent linear functions.  

We can prove by mathmatical that if we only use the linear activation, no matter how many layers or neurons we have, the entire network can be simplified to a single-layer linear regression.  

Linear Algebra: A linear function of a linear function is itself a linear function.  

Common Rule: Don't use linear function for hidden layers // meaningless // equivalent to non-existence
