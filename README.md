## Advanced Learning Algorithms  

The code is based on both C++ and Python API of TensorFlow  

Currently, only the [OpenCL GPU delegate](https://www.tensorflow.org/lite/guide/build_cmake#opencl_gpu_delegate) is supported by the TensorFlow Lite. But perhaps the [VK_KHR_cooperative_matrix](https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VK_KHR_cooperative_matrix.html) will be supported in the future.
